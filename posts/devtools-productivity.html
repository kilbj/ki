我的开发者工具箱 2025

2025-01-28 · 6 min · 工具

工具是开发者的第二双手。经过几年的摸索和迭代，我整理了一份自己每天都在使用的工具清单。

终端 & Shell

  • iTerm2 — 功能丰富的 macOS 终端替代品
  • Oh My Zsh — 插件化 Zsh 配置，效率翻倍
  • fzf — 模糊搜索工具，在终端中快速导航文件、历史命令
  • bat — 带语法高亮的 cat 替代
  • tldr — 简洁的命令行手册，比 man 快多了

编辑器

VS Code 仍然是我的主力编辑器，搭配以下插件：

  • GitLens — 强大的 Git 可视化工具
  • Error Lens — 内联显示错误信息
  • Pretty TypeScript Errors — 让 TS 错误信息更可读
  • GitHub Copilot — AI 辅助编码
"学会你的工具。一个熟练工匠和他的工具之间的边界是模糊的。"

浏览器

Chrome DevTools 是日常必备。几个有用的插件：

  • React Developer Tools — 调试 React 组件
  • JSON Viewer — 格式化 JSON 响应
  • uBlock Origin — 广告拦截，也让页面加载更快

效率工具

  • Raycast — 快捷启动、剪贴板历史、窗口管理
  • Arc Browser — 重新设计的浏览器，分屏和空间管理很棒
  • Obsidian — 本地优先的笔记应用，用于记录技术笔记和个人知识库
  • Warp — 现代化终端（最近在尝试）

DevOps 必备

  • Docker Desktop — 本地容器化开发
  • ngrok — 为本地开发环境生成临时的公开 URL
  • httpie — 更友好的 curl 替代品

工具在不断进化，重要的是定期审视自己的工作流，发现瓶颈并尝试改进。但不要为了换工具而换工具——它能帮你节省时间才是值得的。
