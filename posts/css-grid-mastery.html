CSS Grid 完全掌握：从基础到实战

2025-03-08 · 12 min · CSS

CSS Grid 是布局领域的一场革命。它让我们能用二维的方式思考布局，而不是像 Flexbox 那样在一维上费力组合。

什么是 Grid 布局？

Grid 布局是一种二维布局系统，可以同时控制行和列。最简单的用法是定义一个网格容器：

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 20px;
}

上面的代码创建了一个三列等宽的网格，每列之间和每行之间有 20px 的间距。

关键概念

  • 网格线 (Grid Lines) — 构成网格结构的分界线，从 1 开始编号
  • 网格轨道 (Grid Track) — 两条相邻网格线之间的空间，即行或列
  • 网格单元格 (Grid Cell) — 四条网格线围成的区域
  • 网格区域 (Grid Area) — 任意数量单元格组成的矩形区域

fr 单位

fr 是 Grid 特有的弹性单位，代表可用空间的一份比例：

grid-template-columns: 200px 1fr 2fr;

实战：博客卡片布局

auto-fill 结合 minmax() 是一个极其强大的模式——它让 Grid 自动决定列数，每列最小 320px，最大占满剩余空间。不需要任何媒体查询就能适配从手机到宽屏的各种屏幕。

Grid 和 Flexbox 不是二选一的关系。我的经验是：网格用 Grid，一维排列用 Flexbox。它们可以嵌套使用，各取所长。

Grid 区域命名

对于复杂的页面布局，区域命名让代码一目了然。用 grid-template-areas 配合 grid-area，可以用 ASCII 图的方式定义布局。

总结

CSS Grid 是现代 Web 布局的必备技能。掌握它可以让你的布局代码更简洁、更灵活、更易维护。
